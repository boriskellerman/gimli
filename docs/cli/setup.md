# `gimli setup`

Initialize `~/.gimli/gimli.json` and the agent workspace.

Related:
- Getting started: [Getting started](/start/getting-started)
- Wizard: [Onboarding](/start/onboarding)

## Examples

```bash
gimli setup
gimli setup --workspace ~/gimli
```

To run the wizard via setup:

```bash
gimli setup --wizard
```
